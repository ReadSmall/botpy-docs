# 表情对象

用来表示一个表情基本单位。

# Emoji

字段名类型描述
idstring表情 ID，系统表情使用数字为 ID，emoji 使用 emoji 本身为 id，参考 Emoji 列表
typeuint32表情类型 EmojiType

# EmojiType

值描述
1系统表情
2emoji 表情

# Emoji 列表
