# Botpy简介

# 介绍

Botpy 是基于 机器人开放平台APIopen in new window 实现的机器人框架, 为开发者提供一个易使用、开发效率高的开发框架

# 亮点

  • 轻量，简洁，统一的代码结构，仅8行即可构建一个简单的程序
  • 容易入门，无需学会asyncio也可使用，同时保留较高并发能力
  • 保留官方http API中Json数据的结构字段，带你学习官方结构，日后可自行开发适合自己的SDK
  • 迅速的更新速度，跟上最新潮流
